# Multithreaded Port Scanner

A Python-based TCP port scanner that uses multithreading to scan ports quickly.

## Features
- Multithreaded scanning
- Timeout handling
- Custom port range
- Detect open ports

## Technologies
Python
Socket Programming
Threading

## Usage
python scanner.py
